All Synatax here :


UConsole :

UConsoleWrite("textToWrite") : Echo a message in the terminal

UConsoleWriteColorBash("nameOfColor", "textToWrite") : Echo a message in color (List of color : red,blue,green,yellow,purple,cyan,lightgray,darkgray)

UConsoleInput(variable, "text") : Create a Input for variable

UConsoleSystemBashCreateDirectory("nameOfDirectoryToCreate") : Create a Directory

UConsoleSystemBashDeleteDirectory("nameOfDirectoryToDelete") : Delete a Directory

UConsoleSystemBashCreateFile("nameOfYourCustomFile", "extensionOfYourCustomFile") : Create a File

UConsoleSystemBashDeleteFile("nameOfFileToDelete") : Delete a File

UConsoleSystemBashWriteFile("nameOfFileAndExtension", "textToWriteInFile") : Write in File


USystem :

USystemShellCommand("CommandShell") : Execute Shell Command

USystemListDirFile() : View All File in Directory

USystemMakeArchiveFile("VarArchiveFile", "NameOfArchiveFile", "ExtensionOfArchiveFile") : Make a custom archive file

USystemKernelHash("textToHash") : Hash a text in md5

USystemExecuteShellFile("shellFileToExecute") : Execute Shell File

USystemUNIXInfo() : Get Unix Info

USystemShellCurrentUser() : Get Current User

USystemKernelSudo("commandToSudo") : Sudo a command

USystemKernelSudoApt("commandToSudoApt") : Sudo Apt a command

USystemShellChmod("typeOfChmod", "directoryOfChmodOrFile") : Chmod a File or Directory
